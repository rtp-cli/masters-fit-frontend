MF-004 P0 · color semantics

Completion gets its own signature

Before
Start workout ✓ Completed
success === brand.primary === #0A0A0A. "Completed" is pixel-identical to a primary CTA — the most rewarding moment has no distinct signal.
After
Start workout ✓ Completed
Success keeps ink (monochrome holds) but takes a reserved FORM — outlined ink + check + ring — so completed never reads as a default action. Decoupled into its own --color-success token.
